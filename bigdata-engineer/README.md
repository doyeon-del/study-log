# 빅데이터분석기사 실기

실기 시험 준비 코드 아카이브. 시험은 인터넷 검색이 불가하고 `help()`, `dir()` 정도만 쓸 수 있는 환경이라, 자주 쓰는 패턴을 손에 익히는 것이 목적이다.

## 구성

| 디렉토리 | 작업 유형 | 핵심 |
| --- | --- | --- |
| [`type1_processing/`](type1_processing) | 작업 1유형 | 전처리 · 기술통계 단답 |
| [`type2_modeling/`](type2_modeling) | 작업 2유형 | 분류/회귀 모델링 → 제출 csv 형식 |
| [`type3_statistics/`](type3_statistics) | 작업 3유형 | 가설검정 · 회귀분석 (scipy, statsmodels) |
| [`cheatsheet/`](cheatsheet) | — | 자주 까먹는 문법 정리 (시험장 대비) |

## 사용 라이브러리

pandas, numpy, seaboran, scikit-learn, scipy
