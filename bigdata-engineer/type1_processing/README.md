# 작업형 1유형

T1-1 ~ T1-35 유형을 스킬 묶음별 노트북으로 정리. 각 노트북 셀 주석(`# T1-xx`)이 아래 표와 1:1로 대응한다.

| 노트북 | 묶음 | 포함 |
| --- | --- | --- |
| `A_cleaning.ipynb` | 데이터 정제 (이상치·결측·중복) | T1-1, 2, 3, 15, 23 |
| `B_scaling.ipynb` | 스케일링/변환 | T1-4, 9, 10, 11 |
| `C_descriptive.ipynb` | 기술통계 | T1-5, 13, 16 |
| `D_groupby.ipynb` | 그룹·집계·인덱스 | T1-6, 12, 14 |
| `E_transform.ipynb` | 변형/파생 | T1-7, 8, 20, 21, 24, 31, 32 |
| `F_timeseries.ipynb` | 시계열 | T1-17, 18, 19, 22, 29, 30, 33 |
| `G_string.ipynb` | 문자열 | T1-25, 26, 27, 28 |
| `Z_advanced.ipynb` | 응용 (상/중) | T1-34, 35 |

## 메모

- 데이터는 seaborn/sklearn 내장셋으로 대체해 연습 (별도 파일 없이 실행).
- 문제 지문은 원문 복붙 대신 본인 말로 요약 + 출처 표기.
- 우선순위: ⭐⭐⭐ 밀집 묶음(A·C·D·F·G)부터.
