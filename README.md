# study-log

개인 학습 기록 통합 저장소. 자격증 · 강의 · 코딩 연습 코드를 한곳에 모은다.

## 구성

| 디렉토리 | 내용 |
| --- | --- |
| [`bigdata-engineer/`](bigdata-engineer) | 빅데이터분석기사 실기 준비 코드 |
| _algorithm/_ | (예정) 알고리즘 풀이 |

## 메모

- 학습/연습 목적의 아카이브. 완성된 결과물(포트폴리오)은 별도 레포에서 관리한다.
- 시험 문제 지문 원문은 올리지 않는다 — 유형을 본인 말로 요약하고 풀이 코드 위주로 남긴다.
