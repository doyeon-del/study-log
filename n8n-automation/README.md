# n8n 자동화 학습 트랙

AI를 끼운 자동화 워크플로우를 n8n으로 짜는 법을 공부하는 노트. 설치해서 클릭부터 하기 전에, **무엇이 어떻게 도는지**를 먼저 정리하는 개념 중심 트랙이다.

## n8n을 한 줄로

트리거(언제) → 노드들(무엇을)을 선으로 이어 만드는 오픈소스 워크플로우 엔진. Zapier·Make와 같은 계열이지만 셀프호스팅이 되고, 분기·반복·코드·AI 에이전트를 자유롭게 섞을 수 있다. 노드 사이로는 JSON이 흐르고, 그 JSON을 다음 노드에서 참조하는 게 작업의 핵심이다.

## 왜 이걸 활용하는가?

- 반복 작업(수집 → 정리 → 알림 → 저장)을 코드 한 줄 없이 묶을 수 있다.
- LLM 호출·분류·요약을 파이프라인의 한 단계로 끼워 "AI가 판단하는 자동화"를 만든다.
- 워크플로우 자체가 문서가 되어, 무엇이 도는지 눈으로 설명된다.

## 학습 원칙

- 새 노드를 만나면 "입력 JSON이 뭐고, 출력 JSON이 뭔가"부터 본다. n8n 학습의 80%는 데이터 흐름이다.
- UI는 버전마다 바뀌므로, 메뉴 경로를 외우기보다 **개념과 데이터 모양**을 잡는다. 정확한 노드명/옵션은 [공식 문서](https://docs.n8n.io)에서 확인.
