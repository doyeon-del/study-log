# algorithm

알고리즘 풀이 기록. 프로그래머스 문제를 손으로 설계 → IDE에서 검증·제출 → 통과한 코드를 커밋한다.
SSAFY A형 / 코딩테스트 대비용. **1일 1알고리즘**이 목표.

## 폴더 구조

```
algorithm/
├── programmers/
│   ├── Lv1/        # 프로그래머스 레벨 1
│   ├── Lv2/        # 프로그래머스 레벨 2
│   └── Lv3/        # 프로그래머스 레벨 3
└── _template/
    └── SolutionTemplate.java   # 새 문제 풀 때 복사해서 사용
```

## 파일 네이밍

`programmers/Lv{n}/{문제번호}_{문제명}.java`

예: `programmers/Lv2/12973_짝지어제거하기.java`

## 커밋 메시지 규칙

```
solve: 프로그래머스 Lv2 - 짝지어 제거하기
```

- 한 문제 = 한 커밋 (잔디 + 기록 추적이 깔끔)
- 다시 푼 경우: `revisit: 프로그래머스 Lv3 - 네트워크`

## 풀이 코드 규칙

- 파일 상단에 풀이 메타 주석(문제 링크 · 분류 · 날짜 · 소요시간 · 접근 · 막힌 점)을 꼭 남긴다.
- 막힌 문제는 주석에 `// TODO 복습` 표시 → 주말에 다시 푼다.
